# image-detection

> [USAGE] img-detection COMMAND [-o output_format] input_filename

>   COMMAND
> 
>     binary    二値化
> 
>     convert   ファイルフォーマット変換
> 
>     rectangle 検出された線の位置に矩形を書き込む
> 
>     separate  検出された線を個別ファイルに書き出す
> 
> 
>   -o output_format
> 
>     gif [default]
> 
>     jpg
